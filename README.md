Ce projet est une page web réalisée dans le cadre du module Bases de données / HTML-CSS.
L’objectif était de créer un site complet, responsive et structuré autour d’un thème libre : ici, une fanpage dédiée à Cristiano Ronaldo.

Le site comporte plusieurs sections accessibles depuis la barre de navigation :

- Accueil : grande bannière d’introduction et trois cartes thématiques.
- Biographie : présentation de la carrière à travers des images et textes dynamiques.
- Clubs : galerie interactive présentant les différents clubs de sa carrière.
- Palmarès : mise en avant des principaux trophées et statistiques.
- Moments légendaires : grille regroupant quatre actions iconiques.
- Galerie : collection d’images liées à sa carrière et ses distinctions.
- Contact : formulaire simple permettant d’envoyer un message.

Les technologies que j'ai utilisées :
- HTML5
- CSS3
- Google Fonts
- Font Awesome

Projet réalisé dans le cadre du module Bases de données (HTML/CSS).
